Installing

1. If you are using a Mac, the easiest choice is to use the installer available for the Mac.

2. If you wish to install manually, or you are on the Windows platform, Move this folder (the "Jamoma" folder) into the following folder:

  • Mac OSX: /Library/Application Support/Cycling '74/
  • Win XP: C:\Program Files\Common Files\Cycling '74\

3. If you have downloaded Jamoma from the Subversion (SVN) repository and are using MacOS X: Locate the zipped file Jamoma/library/externals/build_mac.zip and unzip it.

4. If you are using Windows, copy the file jamoma-init.txt to the C:\Program Files\Common Files\Cycling '74\init\ folder. DO NOT DO THIS if you on a Mac - Jamoma will not work

5. Check out the Jamoma documentation.

 

Requirements

Jamoma requires Cycling '74's Max, version 4.5.7 or newer.

Audio modules require Cycling '74's MSP.

Video modules require Cycling '74's Jitter.

Ambisonic modules require both Cycling '74's MSP and Jitter.

 

Examples

To get started, there is an examples folder in the Jamoma/documentation folder. Trond Lossius has also created some excellent tutorials. Also, if you are interested in creating your own modules, be sure to check out the JIG (Jamoma Interface Guidelines) which is in the documentation folder.
