jmod.oscnet

Send and receive OpenSoundControl over a network port. You may receive directly or join up to 2 multicast addresses in addition to receiving directly.

Parameters

Name Type Range Clipmode Ramp Enabled Repetitions Description
/clock/set msg_symbol default none no yes Sync module to external clock.
/set_clock max set module to use the Max scheduler (default behaviour).
/mode toggle default none no yes Choose between single IP or multicast usage
/multicast_ip generic default none no no Define the multicast host to for OpenSoundControl messages to which you would like to join. This may be symbolic or an ip address.
/multicast_join msg_int default none no yes Set the network port number on which to receive OpenSoundControl messages.
/multicast_port msg_int default none no yes Set the network port number used for sending out multicast OpenSoundControl messages.
/osc_prefix generic default none no no Define a string to concatenate to the beginning of all OpenSoundControl strings sent by this module.
/receive_port msg_int default none no yes Set the network port number on which to receive OpenSoundControl messages.
/send_host generic default none no no Set the host to which OpenSoundControl messages should be sent. This may be symbolic or an ip address.
/send_port msg_int default none no yes Set the network port number used for sending out OpenSoundControl messages.
/ui/freeze toggle default none no yes Turn off the updating of user interface elements when parameters change. This may be done to conserve CPU resources.
