Parameters

Name Type Range Clipmode Ramp Enabled Repetitions Description
fullscreen msg_toggle 0 1 none none 1 Toggle fullscreen - can be turned on and off with the ESC key.
interpolate msg_toggle 0 1 none none 1 Interpolate the video output
video/bypass msg_toggle 0 1 none none 0 Bypasses the video processing algorithm in the module - allowing unaltered video through.
video/freeze msg_toggle 0 1 none none 0 Freezes the last frame of output from the module's processing.mxtorithm.
video/mute msg_toggle 0 1 none none 0 Mutes (turns off) the module's.mxtorithm - generating no output.
video/preview msg_toggle 0 1 none none 1 Turns on/off the video display in the module's preview window.

About Jamoma

Jamoma is a system for creating and exchanging structured Max patches. It consists of both a set of guidelines and an implementation of those guidelines. For more information please visit jamoma.org.
