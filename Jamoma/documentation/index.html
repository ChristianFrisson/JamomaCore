Jamoma
A modular standard for Max, MSP, and Jitter

Index of Documentation

Welcome

This is an overview of the available documentation for Jamoma. The documentation should be growing steadily as the project gets under way. Please submit suggestions, additions, etc. to one of the project developers or to the jamoma-devel list that is used as a part of this project. Information on the jamoma-devel list may be found at the Jamoma Project page - linked at the bottom of this document.


Using Jamoma Modules

Using Jamoma Modules is a guide to getting started with using modules in Max, MSP, and Jitter. This is for users of modules, rather than developers of modules.


Jamoma Style Sheet

The primary documentation for Jamoma is the Jamoma Style Sheet. This document contains the specifications for all Jamoma (or Jamoma-Compliant) modules.
