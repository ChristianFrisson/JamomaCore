input
Audio Input

The input module provides 2 channels of audio input from your soundcard/adc and provides facilities for alternatively playing back from a soundfile or test source. This module should be installed in the input location.

size = n/a

Connections:
I/O Type Additional Info
output(1) audio n/a
output(2) audio n/a

Instructions:
Name Arguments Ramp Enabled Description
source_select int no Chooses the sound source to send to the outlets. 0 = No Input, 1 = Live Input, 2 = SoundFile Input, 3 = test source: 1KHz sine wave, 4 = test source: pink noise.
play toggle no Turns on/off the soundfile playback.
loop toggle no Turns on/off looping for the soundfile playback.
direct instructions no Direct access to the underlying sfplay~ object. For more info on sfplay~, see the MSP2 reference manual.
gain float yes Sets the gain for channels 1 & 2 in decibels (where 0dB is full range).
gainR float yes Sets the gain for channel 2 in decibels (where 0dB is full range) when the controls are not linked (see the "link" instruction below).
midigain int yes Sets the gain for channels 1 & 2 in MIDI units, where 127 is full range.
midigainR int yes Sets the gain for channel 4 in MIDI units, where 127 is full range, when the controls are not linked.
gainInc n/a no Increases the midigain for for channels 1 & 2 by one.
gainDec n/a no Decreases the midigain for for channels 1 & 2 by one.
link toggle no Gangs the left and right gain sliders together when toggled on.

Feedback:
Name Arguments Description
<instruction> <arguments> All instructions for this module are returned.
