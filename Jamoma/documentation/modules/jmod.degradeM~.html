degradeM
Digital Distortion

The degradeM module creates digital down-sampling effects. This module responds to all instructions provided by Jade's audio libraries.

size = single x single

Connections:
I/O Type Additional Info
input(1) audio n/a
output(1) audio n/a

Instructions:
Name Arguments Ramp Enabled Description
sr-ratio float yes Sets the ratio to the current sample rate. 1.0 = the current sample rate, 0.5 = half the current sample rate, etc.
bitdepth int yes Sets the bitdepth to down-sample to.

Feedback:
Name Arguments Description
<instruction> <arguments> All instructions for this module are returned.
