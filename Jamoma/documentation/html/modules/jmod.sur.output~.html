Parameters

Name Type Range Clipmode Ramp Enabled Repetitions Description
offset msg_int 0 32 both 1 Set channel offset for the multicable. Changes only take effect when audio is off.
setup msg_list N/A none none 1 List of output channels that multicable signal is routed to.
voices msg_int 0 32 both 1 Set number of output channels to use. Changes only take effect when audio is off.
