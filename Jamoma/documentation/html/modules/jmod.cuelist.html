Configuration

Module Type: control
Algorithm Type: default

Interface Size: 1U-half

Number of signal inlets: 0
Number of signal outlets: 0

Parameters

Name Type Range Clipmode Ramp Enabled Repetitions Description
listen msg_toggle 0 1 none none 0 Listen to remote change of parameters.

 

Messages

Name Type Range Clipmode Ramp Enabled Repetitions Description
cue msg_generic 0.000000 1.000000 none none 1 Recall cue event from cue script.
documentation/help msg_none N/A none none 1 Open the help patch for the module.
documentation/html msg_none N/A none none 1 Open the online html reference for this module.
get_state msg_generic 0.000000 1.000000 none none 1 Poll state for all bridged modules, and display in text window.
load_script msg_generic 0.000000 1.000000 none none 1 Load cue script from file.
NOTE: jcom.cue_list will be watching the file and automatically update if the content of the file is changed. This way you can use an external editor to work on the cue script while Jamoma is running.
module/view_internals msg_none N/A none none 1 Attempts to open the internal algorithm for viewing. This works for most modules. Some modules may choose to cloak the algorithms - preventing this message from functioning.
preset/default msg_none N/A none none 1 Open the default preset file and recall the first preset in that file.
preset/load msg_symbol N/A none none 1 Open an xml-preset file and recall the first preset in that file. An optional argument defines the file to open.
preset/recall msg_float 0.000000 1.000000 none none 1 Recall a preset by number - you can also choose presets from the module menu.
preset/save msg_symbol N/A none none 1 Write an xml-preset file to disk. An optional argument defines the file to open.
preset/store msg_int 0 1 none none 1 Store a preset by number in memory. All presets present in memory will be written to disk when you send a save_settings message to the module.
remote msg_generic 0.000000 1.000000 none none 1 Wireless communication of messages to modules via the bridge.
save_script msg_generic 0.000000 1.000000 none none 1 Save cue script to a file. An argument determines the path and name of the file. No argument will bring up a file dialog (like saveas) or save the script to the existing file that is in use.
view msg_generic 0.000000 1.000000 none none 1 View current cue script in a text window.

 

Return

Name Type Range Clipmode Ramp Enabled Repetitions Description
cues msg_generic 0.000000 0.000000 none 1 Returns a list of all cues in the script.
remote msg_generic 0.000000 0.000000 none 1 Reports all changes of values in all modules

 

About Jamoma

Jamoma is a system for creating and exchanging structured Max patches. It consists of both a set of guidelines and an implementation of those guidelines. For more information please visit jamoma.org.
