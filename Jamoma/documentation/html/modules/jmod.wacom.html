Parameters

Name Type Range Clipmode Ramp Enabled Repetitions Description
/clock/set msg_symbol default none no yes Sync module to external clock.
/set_clock max set module to use the Max scheduler (default behaviour).
/menu menu default none no yes Choose which wacom device to use
/nopointer toggle default none no yes Move mouse pointer
/on toggle default none no yes Turn polling on
/ui/freeze toggle default none no yes Turn off the updating of user interface elements when parameters change. This may be done to conserve CPU resources.

 

Messages

Name Type Range Clipmode Ramp Enabled Repetitions Description
/documentation/help generic n/a none no yes Open the help patch for the module.
/documentation/html generic n/a none no yes Open the online html reference for this module.
/init generic default none no yes Initialization of wacom object
/module/init generic n/a none no yes Initialize this module.
/module/parameter_names/get n/a n/a none no yes Returns a dump of symbols out the feedback outlet with the names of parameters accessible in this module.
/module/size/get generic default none no yes Get size of module.
/module/view_internals n/a n/a none no yes Attempts to open the internal algorithm for viewing. This works for most modules. Some modules may choose to cloak the algorithms - preventing this message from functioning.
/preset/default n/a n/a none no yes Open the default preset file and recall the first preset in that file.
/preset/direct generic n/a none no yes Access directly to the pattrstorage object. See Max Reference.
/preset/load msg_symbol n/a none no yes Open an xml-preset file and recall the first preset in that file. An optional argument defines the file to open.
/preset/recall msg_float default none no yes Recall a preset by number - you can also choose presets from the module menu.
/preset/save msg_symbol n/a none no yes Write an xml-preset file to disk. An optional argument defines the file to open.
/preset/store msg_int default none no yes Store a preset by number in memory. All presets present in memory will be written to disk when you send a save_settings message to the module.
/ui/refresh msg_bang default none no yes Update displayed values for module to reflect current state.

 

Return

Name Type Range Clipmode Ramp Enabled Repetitions Description
/buttons msg_int 0 3 low N/A yes buttons of the Wacom tool
/buttons/raw msg_int 0 5 both N/A yes button flags of the Wacom tool (odd when pressed, even when released)
/direction msg_int 0 1 both N/A yes Direction of the Wacom tool
/module/size generic default none N/A yes Size of module.
/position/x msg_float 0.000 1.000 both N/A yes x position of the Wacom tool
/position/y msg_float 0.000 1.000 both N/A yes y position of the Wacom tool
/preset/out generic default none N/A yes returns the ouptput of pattrstorage object
/pressure msg_float 0.000 1.000 both N/A yes pressure of the Wacom tool
/proximity msg_int 0 1 both N/A yes Detects the presence of a tool near the tablet (without contact)
/rotation msg_float 0.000 1.000 both N/A yes rotation of the Wacom tool
/tilt/x msg_float 0.000 1.000 both N/A yes x tilt of the Wacom tool
/tilt/y msg_float 0.000 1.000 both N/A yes y tilt of the Wacom tool
/touch toggle 0 1 both N/A yes Indicates if the tool touches the tablet

 

About Jamoma

Jamoma is a system for creating and exchanging structured Max patches. It consists of both a set of guidelines and an implementation of those guidelines. For more information please visit jamoma.org.
