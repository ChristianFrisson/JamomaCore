jmod.output~

A generic module for stereo audio output. Includes saturation , limiter and recording device

Table of Contents | Index of Modules | Jamoma.org

Configuration

Module Type: audio
Interface Size: 2U

Number of signal inlets: 2
Number of signal outlets: 0

Parameters

Name Type Range Clipmode Ramp Enabled Repetitions Description
/ch_1 msg_int default none no yes Set logical output for left channel.
/ch_2 msg_int default none no yes Set logical output for right channel.
/disable_ui_updates toggle default none no yes Turn off the updating of user interface elements when parameters change. This may be done to conserve CPU resources.
/gain/midi msg_float 0 158 both yes yes Set master gain as MIDI value.
/limiter/bypass toggle default none no yes Limiter: Active or bypassed.
/limiter/lookahead msg_int 0 256 both yes yes Limiter: The number of samples to look ahead.
/limiter/mode menu default none no yes Limiter: Set the function to be used for calculating the scaling.
/limiter/postamp msg_float -24 48 none yes yes Limiter: Postamp as dB.
/limiter/preamp msg_float 0 48 none yes yes Limiter: Preamp as dB.
/limiter/release msg_float 0 3000 low yes yes Limiter: Set release time as ms.
/limiter/threshold msg_float -24 48 none yes yes Limiter: Set threshold as dB.
/record generic default none no yes no info provided
/record/filetype menu default none no yes Recording: Set file type for sound files.
/record/samptype menu default none no yes Recording: Set sample type for sound files.
/saturation/bypass toggle default none no yes Saturation: Active or bypassed.
/saturation/depth msg_float 0 99 both yes no Saturation: Depth.
/set_clock msg_symbol default none no yes Sync module to external clock.
/set_clock max set module to use the Max scheduler (default behaviour).

 

Messages

Name Type Range Clipmode Ramp Enabled Repetitions Description
/dsp_status generic default both no yes Open DSP Status window.
/gain msg_float -144 0 none yes yes Set master gain as dB value.
/gain/dec generic N/A none no yes Increase master gain MIDI value by one.
/gain/inc generic N/A none no yes Decrease master gain MIDI value by one.
/get_parameter_names n/a n/a both no yes returns a dump of symbols out the feedback outlet with the names of parameters accessible in this module
/help generic n/a both no yes open the online html reference for this module
/load_settings msg_symbol n/a both no yes open an xml-preset file and recall the first preset in that file. An optional argument defines the file to open
/preset_recall msg_int default both no yes Recall a preset by number - you can also choose presets from the module menu
/restore_defaults n/a n/a both no yes Open the default preset file and recall the first preset in that file.
/save_settings msg_symbol n/a both no yes write an xml-preset file to disk. An optional argument defines the file to open
/view_internals n/a n/a both no yes Attempts to open the internal algorithm for viewing. This works for most modules. Some modules may choose to cloak the algorithms - preventing this message from functioning
