input
Audio Input

The input module provides 2 channels of audio input from your soundcard/adc and provides facilities for alternatively playing back from a soundfile or test source.

size = n/a

Connections:
I/O Type Additional Info
output(1) audio n/a
output(2) audio n/a

Instructions:
Name Arguments Ramp Enabled Description
source_select int no Chooses the sound source to send to the outlets.
0 = No Input
1 = Live Input
2 = SoundFile Input
3 = test source: 1KHz sine wave
4 = test source: pink noise.
set_ch_1 int no Sets the logical input channel for live input channel 1
set_ch_2 int no Sets the logical input channel for live input channel 2
play toggle no Turns on/off the soundfile playback.
loop toggle no Turns on/off looping for the soundfile playback.
open filename no Open soundfile. The soundfile has to either be in the search path, or provided with full file path. If no argument is provided, a soundfile can be selected from an open file dialog window.
Mono soundfiles will be played back panned center.
direct instructions no Direct access to the underlying sfplay~ object. For more info on sfplay~, see the MSP2 reference manual.
gain float yes Sets master gain level in decibels (where 0dB is full range).
gain_midi float yes Sets master gain level in MIDI units.
gain_dec n/a no Decrease master gain level by one MIDI unit.
gain_inc n/a no Increase master gain level by one MIDI unit.
gain_1 float yes Sets Channel 1 gain level in decibels (where 0dB is full range).
gain_1_midi float yes Sets Channel 1 gain level in MIDI units.
gain_1_dec n/a no Decrease Channel 1 gain level by one MIDI unit.
gain_1_inc n/a no Increase Channel 1 gain level by one MIDI unit.
gain_2 float yes Sets Channel 2 gain level in decibels (where 0dB is full range).
gain_2_midi float yes Sets Channel 2 gain level in MIDI units.
gain_2_dec n/a no Decrease Channel 2 gain level by one MIDI unit.
gain_2_inc n/a no Increase Channel 2 gain level by one MIDI unit.
link toggle no Link gain levels for Channel 1 and 2 when toggled on.

Feedback:
Name Arguments Description
<instruction> <arguments> All instructions for this module are returned.
