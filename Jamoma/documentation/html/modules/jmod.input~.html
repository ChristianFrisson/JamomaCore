jmod.input~

A generic modul for audio input

Table of Contents | Index of Modules | Jamoma.org

Configuration

Module Type: audio.no_panel
Interface Size: 2U-half

Number of signal inlets: 0
Number of signal outlets: 2

Parameters

Name Type Range Clipmode Ramp Enabled Repetitions Description
/ch_1 msg_int default none no yes Set logical input for left channel.
/ch_2 msg_int default none no yes Set logical input for right channel.
/disable_ui_updates toggle default none no yes Turn off the updating of user interface elements when parameters change. This may be done to conserve CPU resources.
/gain/midi msg_float 0 158 both yes yes Set master gain as MIDI value.
/gain_1/midi msg_float 0 158 both yes yes Set left channel gain as MIDI value.
/gain_2/midi msg_float 0 158 both yes yes Set right channel gain as MIDI value.
/link toggle default none no yes Link gain levels for left and right channels.
/loop toggle default none no yes Loop soundfile.
/play generic default none no yes Play sound file.
/set_clock msg_symbol default none no yes Sync module to external clock.
/set_clock max set module to use the Max scheduler (default behaviour).
/source_select menu default none no yes Select what audio source to use.

 

Messages

Name Type Range Clipmode Ramp Enabled Repetitions Description
/direct generic default none no yes container for various messages sent directly to the sfplay~ object. Refer to the MSP documentation for further details.
/dsp_status generic default none no yes Open DSP Status window.
/gain msg_float -144 0 none yes yes Set master gain as dB value.
/gain/dec generic N/A none no yes Increase master gain MIDI value by one.
/gain/inc generic N/A none no yes Decrease master gain MIDI value by one.
/gain_1 msg_float -144 0 none yes yes Set left channel gain as dB value.
/gain_1/dec generic N/A none no yes Increase left channel gain MIDI value by one.
/gain_1/inc generic N/A none no yes Decrease left channel gain MIDI value by one.
/gain_2 msg_float -144 0 none yes yes Set right channel gain as dB value.
/gain_2/dec generic N/A none no yes Increase right channel gain MIDI value by one.
/gain_2/inc generic N/A none no yes Decrease right channel gain MIDI value by one.
/get_parameter_names n/a n/a none no yes Returns a dump of symbols out the feedback outlet with the names of parameters accessible in this module.
/getmodulesize generic default none no yes Get size of module.
/help generic n/a none no yes Open the online html reference for this module.
/load_settings msg_symbol n/a none no yes Open an xml-preset file and recall the first preset in that file. An optional argument defines the file to open.
/preset_recall msg_int default none no yes Recall a preset by number - you can also choose presets from the module menu.
/restore_defaults n/a n/a none no yes Open the default preset file and recall the first preset in that file.
/save_settings msg_symbol n/a none no yes Write an xml-preset file to disk. An optional argument defines the file to open.
/sf_open msg_symbol default none no yes Open sound file specified by filename. Without a filename a standard open file dialog allows you to choose a file.
/view_internals n/a n/a none no yes Attempts to open the internal algorithm for viewing. This works for most modules. Some modules may choose to cloak the algorithms - preventing this message from functioning.

 

Return

Name Type Range Clipmode Ramp Enabled Repetitions Description
/modulesize generic default none N/A yes Size of module.

About Jamoma

Jamoma is a system for creating and exchanging structured Max patches. It consists of both a set of guidelines and an implementation of those guidelines. For more information please visit jamoma.org.
