video.edge
Glowing Edge Filter

The video.edge module is an edge detection and glowing-edge effects filter. This module responds to all instructions provided by Jade's video libraries.

size = half x single

Connections:
I/O Type Additional Info
input (1) video n/a
output (1) video n/a

Instructions:
Name Arguments Ramp Enabled Description
algorithm symbol no Selects the edge detection algorithm to use. The options are Sobel, Prewitt, and Robcross.
thresh float yes Sets a threshold for the edge gradient.

Feedback:
Name Arguments Description
<instruction> <arguments> All instructions for the module are returned.
