jamoma

a modular standard for max, msp, and jitter

 

 

Welcome

This is an overview of the available documentation for Jamoma. This documentation will be growing as the project moves toward a fully developed version 1.0. Please submit suggestions, additions, etc. to one of the project developers at the jamoma-users list that is used as a part of this project.

If you are interested in who has contributed to Jamoma, we also maintain a credits page with this documentation.

Documentation

Using Jamoma Modules

This is a guide to getting started with using modules in Max, MSP, and Jitter. This is for users of modules, rather than developers of modules.

  1. Using Jamoma modules
    • Tutorial one - a stereo effect module

Creating New Modules

This is a set of tutorials for how to create your own custom modules. It is recommended that you start out with the introduction and the simple control module. You can then skip to the audio or video part depending on your needs and interests.

  1. Introduction
  2. Creating control modules
    • jmod.qmetro.mod - a simple control module
  3. Creating audio modules
    • jmod.???~.mod - a simple audio effect module
  4. Creating video modules
    • jmod.fastblur%.mod - a simple video effect module

Documentation of Existing Jamoma Modules

HTML documentation of all existing Jamoma modules can be found here. You'll also find help files for all modules that comes as part of Jamoma.
