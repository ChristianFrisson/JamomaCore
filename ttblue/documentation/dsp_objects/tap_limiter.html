tap_limiter
Lookahead Limiter
This object features both mono and stereo dsp loops.
Attributes
    k_threshold The threshold above which the signal should be scaled. Specified in decibels (0 dB = full scale, -6 dB = 0.5, etc.).
    k_preamp An amount to amplify the signal by prior to entering the limiter. Specified in decibels (0 dB = full scale, -6 dB = 0.5, etc.).
    k_postamp An amount to amplify the signal by after being processed by the limiter. Specified in decibels (0 dB = full scale, -6 dB = 0.5, etc.).
    k_mode The response shape for the limiter (linear or exponential). The attribute should be set with one of the following selectors:
      k_mode_exponential [default]
      k_mode_linear
    k_lookahead The number of samples the limiter will look ahead to calculate the dynamic change. A latency of this many samples will be introduced [max:255, default: 100].
    k_release The release time of the limiter in milliseconds.
Methods
    clear() Use to reset the buffers other feedback storage. Handy in the event of something blowing-up.
