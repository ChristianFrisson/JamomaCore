tap_mixer_mono
A basic mixer for mixing N mono channels down to 1 output channel
Attributes
    k_channel_gain This selector takes two arguments rather than the usual one argument. The first argument specifies the channel to set the gain for (starting with channel 0), and the second argument specifies the gain in decibels (0 dB = full scale, -6dB = 0.5, etc.).
    k_master_gain Controls a master gain on the sum of the input channels specified in decibels (0 dB = full scale, -6dB = 0.5, etc.).
