tap_buffer_record
Record to an audio buffer
This object features straight-forward recording to a buffer, and also sports a fancy record mode whereby newly recorded samples are faded into the buffer to for smooth sound.
Attributes
    k_mode You can choose the operational mode of the recorder (see the above comment). The attribute should be set with one of the following selectors:
      k_mode_direct [default]
      k_mode_fadein
    k_record Toggle for turning the record "button" on/off.
    k_loop Toggles on/off loop recording.
Methods
    set_buffer(tap_buffer *newbuffer) Pass a pointer to a tap_buffer object to have this object reference that buffer.

    Be sure you don't delete the buffer while this object is still recording to it! Resizing the buffer while this object is recording should not present any problems.
    get_buffer() Returns a pointer to the tap_buffer object that this object is currently referencing.
