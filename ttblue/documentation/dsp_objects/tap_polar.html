tap_polar
Conversion between polar and cartesian coordinates
The primary use of this object will be for FFT-based spectral applications.
Attributes
    k_mode The response shape for the limiter (linear or exponential). The attribute should be set with one of the following selectors:
      k_mode_exponential [default]
      k_mode_linear
    DSP Loop
    void dsp_vector_calc(
            sample_vector in1,
            sample_vector in2,
        short vector_size,
        sample_vector out1
        sample_vector out2
    )
    The values of the inputs and outputs are dependent upon the mode attribute..
