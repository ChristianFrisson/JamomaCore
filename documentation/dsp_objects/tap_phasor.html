tap_phasor
Cycling ramp generator
This object generates a cycling linear ramp from 0.0 to 1.0.
Attributes
    k_phase The current value in the ramp. Thus 0.0 = 0 degrees phase, 1.0 = just-less-than 360 degrees, etc.. [default: 0.0].
    k_frequency The frequency of the oscillator. This is an alternative to the k_ramp_samps or k_ramp_ms attributes below.
    k_ramp_ms The amount of time for the ramp to take in getting to the destination value, specified in milliseconds. This will also flag the object to begin ramping the next time the dsp loop is called.
    k_ramp_samps The amount of time for the ramp to take in getting to the destination value, specified in samples. This will also flag the object to begin ramping the next time the dsp loop is called.
