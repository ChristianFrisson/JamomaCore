DSP Loop
void dsp_vector_calc(
    sample_vector in,
     short vector_size,
    sample_vector out1
    sample_vector out2
)
Out1 is the number of zero-crossings in the most recently completed analysis frame. Out2 is a vector of boolean values (zero or one) which indicate whether a zero-crossing occured at the sample location in the vector.
