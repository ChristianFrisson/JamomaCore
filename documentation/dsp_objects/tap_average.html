tap_average
Running average of a signal
Attributes
    k_mode The averaging method. There are three of these available. The attribute should be set with one of the following selectors:
      k_mode_absolute [default]
      k_mode_bipolar
      k_mode_rms
    k_interval The number of samples over which to average [maximum: 256].
Methods
    clear() Use to reset the buffers other feedback storage. Handy in the event of something blowing-up.
