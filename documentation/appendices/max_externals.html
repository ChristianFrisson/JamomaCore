Writing Max externals with Tap.Tools
Max-specific API documentation
You can write objects for Cycling '74's Max/MSP environment using the Tap.Tools DSP library. In fact, the library is optimized for this purpose and includes a TapTools2 library/dll that you can link to (located in the taptools_maxlib folder).

Examples can be found the Max_Projects folder.

The taptools_max class
    taptools_max::init() Initializes the various static (class) variables.
    taptools_max::getlicense() Returns Tap.Tools licensing information.
